To fetch an already finetuned checkpoint for the EquiformerV2 model run in this folder:

curl https://sid.erda.dk/share_redirect/bSpFYRmDeK -o AI2PR-dft-IS2RE31M.pt

The original pre-trained checkpoint from which the finetuning was performed can be fetched with:

curl https://dl.fbaipublicfiles.com/opencatalystproject/models/2023_06/oc20/s2ef/eq2_31M_ec4_allmd.pt -o eq2_31M_ec4_allmd.pt
